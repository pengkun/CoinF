基本信息

  • rest api: http://140.143.250.209

  • header 必须携带

    1.  BuildVersion:各版本对应的数字，随版本增加，例如：120
2.  SysVersion:手机系统，例如：7.0
3.  Channel:下载渠道号，例如：web，AppStore
4.  Mobile-Model:手机型号，例如：SM-G9550, iPhone6
5.  UUID:APP唯一值，例如 xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
6.  Platform:操作系统，例如 android、ios
7.  Network:网络类型：例如wifi
8.  Language:语言zh_CN,en_US

获取糖果数量

  • /candy/get-count
  • GET

  • parameters:

    1. token - 登录用户的token凭证
2. state - 状态 (只设置成1)

  • 例如：http://140.143.250.209/candy/get-count?token=397cd6d4ceca0bcbed9a822b07c0a8ad&state=1

  • Response:

    {
    "code":1,
    "data":{
        "list":[
            {
                "candy_name_id":"2", // 糖果id
                "sum":"0.29367", // 糖果数量
                "candy_name":"bitcore", // 糖果名字
                "img":"https://coin-1255865046.cos.ap-beijing.myqcloud.com/candy/bitcore.png" // 糖果icon
            }
         ]
    }
}

获取糖果

  • /candy/pickup
  • GET

  • parameters:

    1. token - 登录用户的token凭证
2. state - 状态 (只设置成1)
3. candy_name_id - 糖果id

  • 例如：http://140.143.250.209/index.php/candy/pickup?token=397cd6d4ceca0bcbed9a822b07c0a8ad&state=1&candynameid=2

  • Response:

    {
    "code":1,
    "data":true
}

注册

  • /user/register
  • POST

  • parameters:

    1. username - 手机号
2. password - 密码（md5加密后）
3. nickname - 昵称
4. invite - 邀请码

  • 例如：http://140.143.250.209/index.php/user/register

  • Response:

    {
    "code":1,
    "data":true
}
